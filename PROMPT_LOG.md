## Задание 3: Установить Rust и создать библиотеку с одной функцией.
**Инструмент:** Qwen Code.
### Промпт 1
**Промпт:** "Cоздай простую Rust-библиотеку, которую можно будет вызывать из Python".

**Результат:** Качественный код с использованием актуального Bound API и тестированием за один промпт. Однако попытка собрать проект привела к серии ошибок: линковщик link.exe не был найден, а зависимости pyo3 требовали обновления для поддержки Python 3.13. После настройки зависимостей и сборки проекта все тесты были пройдены.

### Итого
- Количество промптов: 1
- Что пришлось исправлять вручную: установка и настройка Visual Studio Build Tools (для линковщика link.exe). Ручная правка Cargo.toml для обновления pyo3 до версии 0.23. Настройка изолированного окружения .venv и исправление путей для pytest.
- Время: ~60 мин
## Задание 4: Реализовать на Rust высокопроизводительную библиотеку для обработки изображений (например, изменение размера) и вызвать её из Python
**Инструмент:** Qwen Code.
### Промпт 1
**Промпт:** "Реализуй на Rust высокопроизводительную библиотеку для изменения размера изображения методом ближайшего соседа и вызови её из Python".

**Результат:** Получена качественная реализация структуры ImageResizer, использующая rayon для параллельной обработки строк, что позволяет эффективно задействовать многоядерные процессоры. Интеграция с Python реализована через PyO3.

### Промпт 2
**Промпт:** "Улучши тесты: сделай их параметризованными, убери вывод в консоль (используй assert), добейся полного покрытия кода (включая файловые операции и обработку ошибок)".

**Результат:** Разработан комплексный набор тестов (pytest). Внедрена параметризация для тестирования разных размеров, фикстуры tmp_path для безопасной работы с файлами, а также механизмы визуальной отладки (сохранение контрольных изображений).

### Итого
- Количество промптов: 2
- Что пришлось исправлять вручную: Ручная правка Cargo.toml для обновления pyo3 до версии 0.23. Настройка изолированного окружения .venv и исправление путей для pytest. Чистка кода от коментариев.
- Время: ~60 мин
